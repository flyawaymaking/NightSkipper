# 🌙 NightSkipper — умный пропуск ночи в Minecraft

**NightSkipper** — лёгкий и надёжный плагин для Paper, который автоматически пропускает ночь,  
когда спит **настроенный процент игроков** в мире. Поддерживает **EssentialsX AFK**, **vanish** и **спектатор-режим**.

## ✨ Возможности

✅ **Гибкая настройка миров** — выбирайте, в каких мирах работает пропуск ночи  
✅ **Настраиваемый процент сна** — от 0% до 100% игроков для пропуска ночи  
✅ Поддержка **AFK и Vanish игроков** из EssentialsX  
✅ Игроки в **Spectator** или с правом `essentials.sleepingignored` не учитываются  
✅ Отображение прогресса сна через **ActionBar**  
✅ Кастомные сообщения в чате и ActionBar через MiniMessage  
✅ Защита от повторного пропуска (анти-спам)  
✅ Полностью поддерживает **мульти-миры**  
✅ Команда `/nskipper reload` для перезагрузки конфига без рестарта сервера

## ⚙️ Как работает

1. Игрок ложится спать → всем в мире показывается ActionBar:  
   `Спят: 1/4 | Нужно ещё: 1`
2. Когда указанная доля игроков спит (или AFK) — начинается отсчёт:
    - Появляется сообщение: **"Ночь пропускается..."**
    - Через **~3 секунды** ночь пропускается.
3. Мир переходит в утро, гроза и дождь отключаются.

## 🧩 Permissions

| Permission                   | Описание                                                |
|------------------------------|---------------------------------------------------------|
| `nskipper.admin`             | Позволяет использовать команду `/nskipper reload`       |
| `essentials.sleepingignored` | Игрок не учитывается при подсчёте сна (например, админ) |

## 🔧 Конфигурация

Файл `plugins/NightSkipper/config.yml` автоматически создаётся при первом запуске:

```yml
# Настройки пропуска ночи
enabled-worlds:
  - "world"

# Время начала дня и ночи (в тиках Minecraft)
day-time: 0
night-time: 12516

# Процент игроков для пропуска ночи (0.5 = 50%)
sleep-percentage: 0.5

# Сообщения с поддержкой MiniMessage
messages:
  no-permission: "<red>У вас нет прав для этой команды."
  reload-success: "<green>NightSkipper: конфиг перезагружен!"
  reload-usage: "<yellow>Использование: /nskipper reload"
  night-skipping: "<green>Ночь пропускается..."
  sleep-progress: "<yellow>Спят: <gold>{sleeping}/{total}</gold> | Нужно ещё: <red>{needed}</red>"
```

## 💬 Сообщения игрокам

- В чате:
  > Ночь пропускается...

- В ActionBar:
  > Спят: 2/5 | Нужно ещё: 1

- Сообщения конфигурации можно менять с MiniMessage тегами (`<green>`, `<gold>`, `<red>` и др.)

## 🛠️ Установка

1. Скачайте **последний релиз** из раздела [Releases](../../releases)
2. Поместите `.jar` в папку `plugins/` вашего сервера
3. Перезапустите сервер
4. Настройте конфигурацию в `plugins/NightSkipper/config.yml`
5. Для применения изменений в конфиге используйте команду `/nskipper reload`
6. Наслаждайтесь пропуском ночи 🌄

## 🧠 Совместимость
